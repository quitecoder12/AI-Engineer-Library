# Learning Objectives

After completing this chapter you will:

- Run Python code
- Understand print()
- Execute Python files
- Learn basic syntax

---

# Hello World

Create

```
hello.py
```

Write

```python
print("Hello World")
```

Run

```bash
python hello.py
```

Output

```
Hello World
```

---

# Using uv

```bash
uv run hello.py
```

# Multiple Prints

```python

```

---

# Variables

```python
name = "Ajit"

print(name)
```

---

# Numbers

```python
age = 50

print(age)
```

# Expressions

```python

```

---

# Comments

```python

```

# Common Errors

Missing quote

```python

```

Produces

```
SyntaxError
```
