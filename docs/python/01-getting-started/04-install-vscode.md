# Installing Visual Studio Code

---

# Why VS Code?

Visual Studio Code is one of the most popular editors for Python and AI development.

Advantages:

- Free
- Lightweight
- Excellent Python support
- Git integration
- Jupyter notebooks
- Debugger
- AI extensions

---

# Download

Download Visual Studio Code from the official website.

---

# Install Python Extension

After opening VS Code:

Extensions →

Search

```
Python
```

Install the extension published by Microsoft.
