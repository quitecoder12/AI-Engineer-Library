# Installing Python

## Recommended Version

Use the latest stable release of Python 3.x.

For this library we recommend Python 3.13 or newer when possible.

---

## Windows

1. Download Python from the official Python website.
2. Run the installer.
3. Enable **Add Python to PATH**.
4. Complete the installation.

---

## Verify Installation

Open a terminal and run:

```bash
python --version
```

or

```bash
python3 --version
```

You should see the installed Python version.

---

## Verify pip

```bash
python -m pip --version
```

---

## Verify IDLE (Optional)

Python includes the IDLE editor, though we recommend Visual Studio Code for development.
