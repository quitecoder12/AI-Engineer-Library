# What is Python?

Python is a high-level, interpreted, general-purpose programming language.

It focuses on:

- Readability
- Simplicity
- Productivity

Python lets developers solve problems with fewer lines of code than many other programming languages.

---

# Why was Python created?

Python was created by **Guido van Rossum** in 1991.

His goal was to create a language that was:

- Easy to learn
- Easy to read
- Easy to maintain

Today Python powers millions of applications.

# Python vs Compiled Languages

Python is interpreted.

Languages like C++ and Go are compiled.

Python trades a little execution speed for dramatically higher developer productivity.

For AI engineering, that trade-off is usually worth it because most heavy computation is performed by optimized native libraries underneath Python.

---

# Is Python slow?

This is a common misconception.

Python code itself can be slower than compiled languages, but AI workloads typically rely on optimized libraries (often written in C, C++, or CUDA), so Python acts as the orchestration layer.
