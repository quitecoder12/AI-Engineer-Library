# Python for AI Engineers

Welcome to the Python section of the AI Engineer Library.

This is not a traditional Python tutorial.

It is a complete reference for learning Python specifically for AI, Machine Learning, Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), FastAPI, Streamlit, and Production AI.

---

# Learning Objectives

By the end of this section you will be able to:

- Write modern Python
- Understand Python internals
- Build production applications
- Read AI source code
- Build AI APIs
- Build AI agents
- Work with modern AI libraries

---

# Prerequisites

No Python experience is required.

However, basic programming knowledge will help.
