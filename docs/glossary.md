# Glossary

## AI

Artificial Intelligence.

---

## LLM

Large Language Model.

Examples:

- GPT
- Llama
- Mistral
- Gemini

---

## Token

The smallest unit processed by an LLM.

---

## Embedding

A numerical representation of text used for semantic search.

---

## RAG

Retrieval-Augmented Generation.

A technique that combines document retrieval with language model generation.

---

## Vector Database

A database optimized for storing and searching embeddings.

Examples:

- ChromaDB
- Pinecone
- FAISS
- Weaviate
- Milvus

---

## Agent

Software capable of reasoning, planning, and using tools to complete tasks.

---

## Prompt

Instructions given to an AI model.

---

## Context Window

The maximum amount of information an LLM can process in one request.

---

## Inference

The process of generating predictions or responses from a trained model.
